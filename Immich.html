Immich Self-Hosting Guide

Take control of your memories. This guide will walk you through setting up Immich, a powerful self-hosted photo and video backup solution, on a dedicated server.

Important Note: This guide assumes you have a dedicated server (like an Intel N100 mini PC, Raspberry Pi with external storage, or a home server) running Docker. The Flint 2 router does NOT support Docker, so a separate machine is required for Immich.

What You'll Need

  • A dedicated server with Docker and Docker Compose installed.
  • Basic command-line knowledge.
  • A reverse proxy (e.g., Nginx Proxy Manager, Caddy) if you want external access.
  • A domain name (optional).

Step-by-Step Installation

Step 1: Prepare Your Server

Update and Install Dependencies

Ensure your server's package list is up-to-date and install Git:

sudo apt update
sudo apt upgrade -y
sudo apt install git -y
Step 2: Clone Immich Repository

Download Immich Docker Compose Files

Navigate to your installation directory and clone the Immich example repository:

cd /opt
git clone https://github.com/immich-app/immich-docs.git
cd immich-docs/install
Step 3: Configure Environment Variables

Set Up Your .env File

Copy the example environment file and then edit it to set your desired passwords:

cp .env.example .env
nano .env

Key variables to review:

  • DB_PASSWORD=your_secure_password (Change this!)
Step 4: Create Data Directories

Persistent Storage for Immich

Create directories on your host machine and ensure Docker has permission to write to them:

sudo mkdir -p /mnt/immich/upload
sudo mkdir -p /mnt/immich/database
sudo chown -R 1000:1000 /mnt/immich
Step 5: Edit Docker Compose File

Map Volumes and Configure Ports

Open the docker-compose.yml file and modify the volumes section to point to your new host directories.

nano docker-compose.yml

For immich-server:

volumes:
      - /mnt/immich/upload:/usr/src/app/upload

For database:

volumes:
      - /mnt/immich/database:/var/lib/postgresql/data
Step 6: Start Immich

Launch Docker Containers

Start Immich in the background:

docker compose up -d
Step 7: Access Immich Web UI

Initial Setup

Navigate to http://your-server-ip:3000 in your browser to create your admin account.

Step 8: Set Up Reverse Proxy (Optional)

Secure External Access

In Nginx Proxy Manager, create a Proxy Host pointing your domain to the server IP and port 3000. Enable SSL for security.

Step 9: Mobile App Setup

Backup Photos from Your Phone

  1. Install Immich from the Play Store.
  2. Enter your server URL.
  3. Log in and configure backup settings.

Maintenance Commands

  • Stop: docker compose down
  • Start: docker compose up -d
  • Update: 
    git pull
docker compose pull
docker compose up -d
