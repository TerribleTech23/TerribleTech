Immich Self-Hosting Guide

Take control of your memories. This guide walks you through setting up Immich, a powerful self-hosted photo backup solution, on a dedicated server.

Server Note: This requires a dedicated machine (Mini PC, Pi, etc.) running Docker. Your Flint 2 router handles networking, but Immich needs its own "brain."

Step 1: Update & Dependencies

sudo apt update && sudo apt upgrade -y
sudo apt install git -y

Step 2: Clone Immich

cd /opt
git clone https://github.com/immich-app/immich-docs.git
cd immich-docs/install

Step 3: Configure Environment

Copy the example env file and set your database password:

cp .env.example .env
nano .env

Step 4: Storage Paths

Create directories for your photos and database:

sudo mkdir -p /mnt/immich/upload
sudo mkdir -p /mnt/immich/database
sudo chown -R 1000:1000 /mnt/immich

Step 5: Launch

Start the containers in detached mode:

docker compose up -d

Step 6: Access & Mobile

Open http://your-server-ip:3000 to create your admin account. Then, download the Immich app on your S25 Ultra to begin your first backup.
