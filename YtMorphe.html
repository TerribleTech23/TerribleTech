YouTube Morphine

The premium YouTube experience, built for Android.

Step 01

MicroG-RE

To sign in, you need the MicroG-RE compatibility layer. Install this first to ensure account sync works correctly.

Step 02

Installation via Obtainium

Add the repository MorpheApp/YouTubeMorphine to Obtainium. This allows you to track updates directly from GitHub without manual APK hunts.

Step 03

Features & Setup

Enable SponsorBlock and Return YouTube Dislike in the settings menu. For your S25 Ultra, ensure the OLED Black theme is active to save battery and look sharp.
